📊 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

  GET  /health
    → Check if server is running
    → Expected: {"status":"ok",...}

  GET  /webhooks/mailchimp/health
    → Check Bitrix24 connectivity
    → Expected: {"status":"ok","bitrix24":"connected"}

  POST /webhooks/mailchimp
    → Production webhook from Mailchimp
    → Requires: Webhook signature header

  POST /webhooks/mailchimp/test
    → Test endpoint (development only)
    → Can be called without webhook from Mailchimp
